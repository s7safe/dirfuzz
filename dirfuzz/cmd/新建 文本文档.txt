个基于 Golang 语言编写的 Web 应用程序安全漏洞扫描工具，旨在通过自动化的方式发现 Web 应用程序中的漏洞并生成相应的报告。我具有以下主要功能：

    通过指定的 URL 地址和 Wordlist 进行敏感信息扫描；
    通过指定的 URL 地址和 Fuzz Wordlist 进行路径 Fuzzing 扫描；
    支持扫描结果以 CSV 和 JSON 格式输出；
    支持过滤器对指定的敏感信息进行过滤；
    支持代理和请求头自定义；
    支持并发扫描和多个目标同时扫描；
    支持超时时间、重试次数、响应码等参数的自定义；
    支持对漏洞进行标记，方便后续操作；
    支持内置漏洞扫描器，如 SQL 注入、XSS、敏感信息泄露等；
    支持自定义漏洞扫描器，可根据需要增加新的漏洞扫描模块。

我的代码主要由以下几个模块组成：

    cmd：包含命令行工具代码，主要负责解析用户输入参数、启动扫描器等；
    config：包含配置文件的读取代码，主要负责读取配置文件中的参数，如代理、请求头、超时时间等；
    filter：包含过滤器代码，主要负责对指定的敏感信息进行过滤；
    fuzz：包含 Fuzzing 相关的代码，主要负责生成 Fuzz Wordlist 和进行路径 Fuzzing；
    output：包含输出结果相关的代码，主要负责将扫描结果以 CSV 或 JSON 格式输出；
    scanner：包含扫描器相关的代码，主要负责对目标进行扫描，包括内置漏洞扫描和自定义漏洞扫描；
    wordlist：包含 Wordlist 相关的代码，主要负责读取 Wordlist 文件和生成敏感信息扫描的 URL。
